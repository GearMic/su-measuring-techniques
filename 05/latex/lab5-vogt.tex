\subsection{Constructing a Second Order Stop-Band Filter} \label{stop-band-manual}
We define the frequency response of a stop-band filter whose gain is $0$ at $f = f_0 = \SI{50}{\Hz}$
with a bandwidth of $b = \SI{4}{\Hz}$ at \SI{-3}{\dB} (corresponding to an attenuation of $1/\sqrt(2)$).
The sampling rate is $F_s = \SI{1000}{\Hz}$.

In the space of the z-transform, the frequency $f_0$ corresponds to $\exp(i\Omega_0)$ with
\begin{equation*}
  \Omega_0 = 2\pi \frac{f_0}{F_s} = \frac{\pi}{10}
\end{equation*}
To get the desired attenuation, we place a zero
at this point, i.\,e. we add a factor $(1- \exp(i\Omega_0))$ to the denominator.
Since real signals always contain negative frequencies in the same amount as corresponding positive frequencies,
we also need to attenuate \SI{-50}{\Hz} (corresponding to $-\Omega_0$) in the same way,
so we add a factor $(1 - \exp(-i\Omega 0))$.

Now we need to add the property that the filter's maximum gain is $1$ and that it has the desired bandwidth.
This is achieved by adding a pole (i.e. a factor in the denominator)
that is not on the unit circle but still close to the zero that we added.
It can be parametrized as $r \exp(i\Omega 0)$ (with $r$ real). 
For values of z that are far away, the distance to the pole and the zero will be roughly the same,
so modulus of the numerator and denominator are equal, leading to a gain of $1$.
When z approaches the pole and the zero, there comes a point when it is separated from the
zero by a distance $d \approx (1-r)$ and from the pole by a distance $\approx \sqrt{d+(1-r)} \approx \sqrt{2}(1-r)$.
the ratio of the numerator and denominator is then $1/\sqrt(2)$, so the arc around the unit circle
where the gain is lower than \SI{-3}{\dB} has a length of $2d \approx 2(1-r)$.
The bandwidth of the filter thus depends on the distance $r$ of the pole to the origin.
We require the arc $2d$ to correspond to the bandwidth $b = \SI{4}{\Hz}$:
\begin{align*}
  &2(1-r) = 2d \stackrel{!}{=} 2\pi \frac{b}{F_s}\\
  \implies &r = 1-\frac{b}{F_s}\pi = 1 - \frac{\pi}{250} = 0.98743
\end{align*}
% the bandwidth therefore corresponds to a distance of 2d around the unit circle, and d is related to r by d=1-r,
% where r is the distance of the pole from the origin. 

Again, we at two poles for positive and negative frequencies. With both zeros and poles,
the filter transfer function becomes
\begin{equation*}
  H(z) = \frac{(1-\exp(i\Omega_0)) (1-\exp(-i\Omega_0))}%
              {(1-r\exp(i\Omega_0)) (1-r\exp(-i\Omega_0))}
\end{equation*}
Multiplying out the brackets and then diving through by $z^2$, we get $H(z)$
as a ratio of polynomials in $z^{-1}$:
\begin{align*}
  H(z) = \frac{B(z)}{A(z)} &= \frac{1 - 2\cos(\Omega_0) \cdot z^{-1} + z^{-2}}%
              {1 - 2r\cos(\Omega_0) \cdot z^{-1} + r^2 \cdot z^{-2}} \\
       &= \frac{1 - 1.9021 z^{-1} + z^{-2}}%
              {1 - 1.8782 z^{-1} + 0.9750 z^{-2}}
\end{align*}

This is the form used by the Matlab function \texttt{freqz(b, a, N)} to plot
the transfer function. $b$ and $a$ specify the coefficients in $B(z)$ and $A(z)$
so \texttt{b = [1 -1.9021 1]} and \texttt{a = [1 -1.8782 0.9750]}.
This gives the plot shown in \figref{fig:stopband1-freqz}
\begin{figure}
  \centering
  \includegraphics[width=.9\textwidth]{stopband1-freqz}
  \caption{
    Plot of the transfer function of the second-order stop-band filter,
    created using the Matlab function \texttt{freqz}. The frequency
    range was chosen to zoom in on the relevant features.
    We chose $1000$ samples for better visibility.
  }
  \label{fig:stopband1-freqz}
\end{figure}

The code is shown in \figref{code:stopband1-code}
\begin{figure}[h!]
% \begin{lstlisting}[language=Matlab]
% \begin{lstlisting}[basicstyle=\ttfamily,language=Matlab]
\begin{lstlisting}[basicstyle=\ttfamily\footnotesize,language=Matlab]
  % input parameters
  Fs = 1000;
  f0 = 50;
  b = 4;

  % calculate omega0 and r
  omega0 = f0/Fs * 2*pi;
  r = 1-b/Fs*pi;

  % calculate polynomial coefficients
  b = [1 -2*cos(omega0) 1];
  a = [1 -2*r*cos(omega0) r^2];

  % plot using freqz
  freqz(b, a, 256);
  xlim([0 0.3]);
  subplot(2, 1, 2); xlim([0 0.3]); 
\end{lstlisting}
\caption{
  Matlab code to define and plot the second order stop-band filter.
}
\label{code:stopband1-code}
\end{figure}



\subsection{Second Order Butterworth-Type Stop-Band Filter}
Next we construct a stop-band filter with the same characteristics as before,
but created as a Butterworth filter using the Matlab function \texttt{butter(n, Wn, 'stop')}.
\texttt{n} is the order where \texttt{n = 1} actually corresponds to a second order filter,
and \texttt{Wn} is an array containing the break points normalized to the Nyqvist frequency.
\texttt{'stop'} specifies that it is a stop-band filter.

The transfer function plot can be seen in \figref{fig:stopband2-freqz} and
the code in \figref{code:stopband2-code}.
\begin{figure}[h]
  \centering
  \includegraphics[width=.9\textwidth]{stopband2-freqz}
  \caption{
    Plot of the transfer function of the second-order butterworth-type stop-band filter.
  }
  \label{fig:stopband2-freqz}
\end{figure}
\begin{figure}[h]
\begin{lstlisting}[basicstyle=\ttfamily\footnotesize,language=Matlab]
  % input parameters
  Fs = 1000;
  f0 = 50;
  b = 4;

  % generate filter
  n = 1;
  Wn = [f0-b/2 f0+b/2] / (Fs/2);
  [b, a] = butter(n, Wn, 'stop');

  % plot transfer function
  freqz(b, a, 1000)
  xlim([0 0.3]);
  subplot(2, 1, 2); xlim([0 0.3]); 
\end{lstlisting}
\caption{
  Matlab code to generate and plot a second order butterworth-type stop-band filter.
}
\label{code:stopband2-code}
\end{figure}

The transfer functions of the previous filter and the butterworth-type filter have very similar
characteristics since they were both defined to have the same stop band with the same bandwidth.
Both filters have a jump in phase around the center of the stop-band. The manually defined filter
drops more sharply (in theory reaching $-\infty\,\si{\dB}$) in the center of the stop-band because it
has a zero there, whereas the butterworth-type filter has a finite minimum.



\subsection{Other Filter Types}
There are several more filter types supported by Matlab. Among them are
\begin{itemize}
  \item Chebyshev type I: \texttt{cheby1(n,Rp,Wn,'stop');} \\defined with pass-band ripple \texttt{Rp} in \si{\dB}.
  \item Chebyshev type I:  \texttt{cheby2(n,Rs,Wn,'stop');} \\defined with stop-band attenuation \texttt{Rs} in \si\dB.
  \item Elliptical:  \texttt{ellip(n,Rp,Rs,Wn,'stop');} \\defined with both \texttt{Rp} and \texttt{Rs}.
\end{itemize}
For each of these filters we replace the function \texttt{butter(\dots)} in \figref{code:stopband2-code}  
with one of the above, choosing \texttt{Rp = 2} and \texttt{Rs = 20}.
Also, we again choose \texttt{n = 1} which results in second-order polynomials.
The transfer functions are shown in \figref{fig:stopband3-freqz}.

The Chebyshev type I and elliptical filter are identical (this can be seen by looking at the plots and by inspecting
the polynomial coefficients). When changing the \texttt{Rs} value for the elliptical filter, we see that
the coefficients always stay the same, so at this order and with \texttt{Rp = 2} the stop-band attenuation
actually has no impact on the elliptical filter and it is identical to a Chebyshev type I filter.

The Chebyshev type II filter reaches a much lower gain ($\sim\SI{-50}{\dB}$) than the other two but
also has a much slower rise away from the stop band. In fact, it does not adhere to the provided breaking points
which likely means that the chosen filter order was too low.

\section{Sampling an EKG signal}
Next, we study data from an EKG which has a sampling rate $F_s = \SI{300}{\Hz}$
with $N=1024$ samples, see \figref{fig:ekg-plot-provided}.
\begin{figure}
  \centering
  \includegraphics[width=.9\textwidth]{ekg-plot-provided}
  \caption{EKG signal with noise \cite{manual}.}
  \label{fig:ekg-plot-provided}
\end{figure}

The maximum frequency without aliasing is the Nyqvist frequency,
i.\,e. half of the sampling rate:
\begin{equation*}
  F_\text{max} = \frac{F_s}{2} = \SI{150}{\Hz} 
\end{equation*}
The total sampling time is $T = N/F_s$ and the frequency resolution signal's Fourier transform
would be $\Delta f = \frac{1}{T} = F_s/N \approx \SI{0.293}{\Hz}$.
The frequency with the largest amplitude is likely that of the recurring heartbeat signal.
There are about $3$ beats between samples $0$ and $800$, so the frequency is
\begin{equation*}
  f_H = \frac{3}{800/F_s} = \SI{1.125}{\Hz}
\end{equation*}

The data we are actually provided is different from the provided plot in \figref{fig:ekg2-plot-orig}.
We will analyze the data from \texttt{EKG\_02.wav} which contains $N=82893$ samples at a sampling rate
$F_s = \SI{8000}{\Hz}$ so the data spans $T = N/F_s = \SI{10.36}{\s}$.
It is plotted in \figref{fig:ekg-plot-orig}.
\begin{figure}[h]
  \centering
  \includegraphics[width=.9\textwidth]{ekg2-plot-orig}
  \caption{Plot of the data from \texttt{EKG\_02.wav}.}
  \label{fig:ekg2-plot-orig}
\end{figure}

To analyze the frequency content, we take the signal's fourier transform.
The frequency resolution is $\Delta f = 1/T = \SI{0.0965}{\Hz}$ and the Nyqvist frequency is $4000\Hz$.
The fourier transform is plotted in \figref{fig:ekg2-fft}. 
\begin{figure}[h]
  \centering
  \includegraphics[options]{ekg2-fft}
  \caption{Fourier transform of the data from \texttt{EKG\_02.wav}.}
  \label{fig:ekg2-fft}
\end{figure}

We can see that the dominant frequency is
actually not that of the heartbeat, but the noise at \SI{50}{\Hz}. We can filter out the noise by
applying the stop band filter from Sec.\,\ref{stop-band-manual}.
this is done by doing \texttt{x = filter(b, a, x)} where \texttt{b} and \texttt{a} are the arrays
given in Sec.\,\ref{stop-band-manual}. The filtered signal is shown in \figref{fig:ekg2-plot-filtered}.
There is now much more structure visible because the interfering \SI{50}{\Hz} oscillation is not
there anymore. The most important characteristic of the signal is the sharp drop and then
peak in voltage caused by the heartbeat. This is followed about \SI{0.3}{\s} later by a second,
much smaller peak.
\begin{figure}[h]
  \centering
  \includegraphics[width=.9\textwidth]{ekg2-plot-filtered}
  \caption{Plot of the data from \texttt{EKG\_02.wav} with a \SI{50}{\Hz} spot-band filter applied.}
  \label{fig:ekg2-plot-filtered}
\end{figure}



\section{Conclusion}
In this computational lab, multiple digital filters were constructed using Matlab.
The task was to construct stop-band filters with the same stop band characteristics
using different methods. We constructed a filter by placing specific poles and zeros.
This filter had the best characteristics in terms of attenuation of the stop frequency.
Other filter types used were Chebyshev type I and II and an elliptical filter.

Additionally, an EKG signal was analyzed by looking at its Fourier transform. We could
see a strong contribution at \SI{50}{\Hz} likely caused by the mains voltage.
After filtering this out, we identified two distinct peaks for each heartbeat.
